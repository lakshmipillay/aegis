# AEGIS

AEGIS is an AI governance and observability layer for Gemini applications using Datadog LLM Observability.

## Features
- LLM Firewall
- Hallucination & governance scoring
- Token and cost tracking
- Datadog LLM Observability
- Automatic incident creation

## Run
Deploy on Google Cloud Run and set environment variables:
- GEMINI_API_KEY
- DD_API_KEY
- DD_APP_KEY
